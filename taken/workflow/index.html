Workflow

Hoe ben ik er aan begonnen?

We kregen de opdracht om een website te maken waarop we al onze GIP-opdrachten zullen publiceren. Als hulpmiddeltje mochten we een static site generator gebruiken, hier hadden we nog nooit van gehoord!

Op www.staticgen.com vonden we verschillende static site generators. Ik heb er verschillende uitgetest:

  • Cabin
  • Gulp
  • Harp
  • Hexo
  • Metalsmith
  • Punch
  • Ruby
  • Susi
  • Techy

Bij elke generator had ik wel een ander probleem! Na lang zoeken heb ik uiteindelijk gekozen voor Octopress.

De installatie van Octopress ging niet van een leien dakje; ik heb hiervoor de programmeertaal Ruby moeten installeren, Git (Een source code management systeem), en natuurlijk ook Octopress zelf.

De installatie van Octopress

Op de site van Octopress was de nodige informatie te vinden.

Ik heb deze stappen gevolgd:

1. Installeren van Git

choco install git

2. Installeren van Ruby en Ruby devkit en het juistzetten van de paden

    choco install ruby 
    ruby dk.rb init 
    ruby dk.rb install

3. Installeren van Octopress, de dependencies en het default thema

    git clone git://github.com/imathis/octopress.git octopress
    cd octopress
    gem install bundler
    rbenv rehash
    bundle install
    rake install

4. Installeren van een custom thema

Het default thema is ook maar gewoon normaal, dus heb ik gekozen voor een custom thema (te vinden hier op Github).

In de map van octopress:

 git clone git://github.com/ioveracker/mnml.git .themes/mnml
 rake install['mnml']
 rake generate

Je vorige thema wordt automatisch vervangen door het nieuwe.

Het bewerken van de site: workflow

Maken van een nieuwe pagina

rake new_page["name"]

Bewerken van de navigatiebalk

octopress\source\_includes\custom in de file navigation.html

Preview weergeven van je site

rake preview

Taken schrijven in Markdown

https://stackedit.io/editor

Markdownfiles

octopress\source\taken

Aanpassen html, css, yml, …

Brackets

Kleuren aanpassen van het thema

octopress\.themes\mnml\sass\custom
