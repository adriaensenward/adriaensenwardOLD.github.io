Ondertiteling Van Een Instructiefilmpje

Ik heb de tool Subtitle Edit gebruikt. Onder het filmpje vind je de volledige vertaling.

Dutch version (494 words)

Hallo iedereen, ik ben Austin en vandaag ben ik er met een Steam OS handleiding. Het is volledig mogelijk om voor jezelf een “Steam machine” te bouwen. Dit is de allereerste uitgave dus het is nog een beetje ingewikkeld; voor zover de installatie. Er missen nog een paar grote functies zoals het streamen van games.

Allereerst zal je compatibele hardware nodig hebben in je computer. In dit geval is dat een computer met een NVIDIA grafische kaart en een tamelijk recent moederbord dat UEFI-booting ondersteunt. Steam OS is gebaseerd op Linux maar er is geen mogelijkheid om beide systemen tegelijk te installeren; er is geen dual booting. Dus als je beslist om voor Steam OS te kiezen; houd er rekening mee dat het alle andere bestanden op je harde schijf zal verwijderen, dus wees voorzichtig!

Zodra je klaar bent om te beginnen kan je de Steam OS Installer downloaden van Valve. Deze link is te vinden in de beschrijving van deze video. Je kan ook de hersteloptie gebruiken; deze is zogenaamd gemakkelijker hoewel ik een paar problemen had toen ik deze probeerde. Dus voor deze handleiding blijven we bij de standaardmethode.

Neem er een USB stick bij van op z’n minst een aantal gigabytes. Deze gaan we formatteren, zodat alle bestanden gewist worden. Windowsgebruikers moeten rechtermuisklikken op de USB stick in de Windows verkenner en daarna op “formatteer”. Wees er zeker van dat het op FAT32 staat, noem het “Steam OS” en klik dan op “formatteer’. Mac-gebruikers openen het hulpprogramma voor schijven. Selecteer de USB stick, klik op “verwijder”, kijk opnieuw na dat het op FAT32 staat en noem het “Steam OS” en je bent hier klaar. Zoek daarna het Steam OS zipbestandje en pak al de bestanden uit op je USB stick. Kijk na dat de bestanden niet in een of andere folder staan, ze zouden op de kern van je schijf moeten staan. Tijd om er in te vliegen!

Steek de USB stick in je toekomstige Steam Machine en betreed de BIOS. Normaal gezien met de F10- of DEL-knop. Zoek het bootmenu en schakel de Legacy + UEFI of een generische UEFI optie in. Kijk na dat de USB ingesteld staat als eerste optie bij het opstarten en verlaat de BIOS. Als alles goed gaat zie je dan het Steam OS-logo tevoorschijn komen. Selecteer vanaf hier de automatische installatieoptie en geef het een tijdje om alles te laden op je harde schijf. Het kan zijn dat het een paar keer vasthangt, maar wees geduldig en dan zou het moeten lukken. Wanneer het klaar is zal je dit scherm te zien krijgen; dit betekent dat je veilig je USB stick mag verwijderen en het toegestaan is om de computer te herstarten.

Het zal je dan een algemeen bootmenu laten zien. Dit deel is belangrijk: ga naar het vervolgkeuzemenu en selecteer: “Gnome”. Typ dan “steam” als gebruikersnaam en wachtwoord, beide in kleine letters. Dan zou je een bijna leeg bureaublad te zien krijgen.

English version (509 words)

Hey guys, this is Austin, and today I’m here with the Steam OS tutorial and demo. It’s totally possible to build yourself a Steam machine right away. Over this is the very first release. So it’s a bit complicated as far as the install goes. It is still lacking a few major features like game streaming.

First of all you’ll need compatible hardware in your pc. This means a system with an NVIDIA graphics card, and a fairly recent motherboard that supports UEFI boot. Steam OS is based on Linux, you won’t be able to install it as well, as there is no dual booting. So if you do decide to install Steam OS, keep in mind that It will delete everything that’s on your hard drive. So definitely be careful. Once you’re ready to begin, you need to download Steam OS installer.zip from Valve, and I’ll have that link in the description. You could also use the system restore option which is supposedly easier. However, when I tried it I ran into quite a few issues. So for this tutorial we’re going to stick to the standard method.

Grab yourself an USB drive of at least a couple gigabytes and now we’re going to freshly format it. SMake sure that it’s been whiped clean of any data. On Windows just right click the drive from my computer and select format. Make sure it’s set to FAT32, name it Steam OS and then click “format”. On mac: open up disk utility, pick out your USB drive and select “erase”. Again, make sure it’s set to FAT, name it Steam OS and you’re good here. Now, find the Steam OS installer zip we downloaded earlier, and extract all the files right on to your USB drive. Make sure the files aren’t inside any kind of folder, they should be at the root of your drive. Time to get this started.

Plug the USB into your soon to be Steam machine and enter the BIOS. Usually using the F10, but the delete key has a boot. Find the boot menu and enable “legacy + UEFI” or a generic UEFI option. Then make sure your USB drive is set as the first option at boot and exit the BIOS. If everything worked you should see the Steam OS install screen show up. From here just select the automated install and give it a little while to load everything onto your hard drive. It might appear to freeze a few times, but just be patient and it should complete. Once it’s done, you’ll see this screen. That means it’s safe to unplug the USB drive and allow the pc to reboot.

It will then bring you to a generic looking boot menu. This part is important: go to the drop down menu and select “Gnome”. Then type “steam” as the username and “steam” as the password, both is lowercase, and it should bring you to a mostly empty desktop.
